- **Etapa 3 — La capa competitiva**: el pipeline completo (RADAR → INVESTIGADOR → SALA DE REDACCIÓN → MOTOR SEO → TAXÓNOMO → COMPUERTAS → PUBLICADOR), con las tres Compuertas como sistema inmunológico, modos reales con degradación por sensibilidad, cadencia de publicación real y Sala de Revisión.
