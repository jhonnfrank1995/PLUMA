```
        DIVERGIR                    CONSTRUIR              CONVERGER
  ┌───────────────────┐      ┌──────────────────┐   ┌──────────────────┐
  │ First Principles  │      │  CTO Mode        │   │ Critical Review  │
  │ Product Vision    │  →   │  Design          │ → │ Decision         │
  │ Innovation Engine │      │  Excellence      │   │ Framework        │
  │ Future Thinking   │      └──────────────────┘   └──────────────────┘
  └───────────────────┘

```

## Regla de proporción
Puerta de dos direcciones (reversible barato) → decide rápido, salta el ritual.
Puerta de una dirección (cara de revertir) → stack completo + registro.
