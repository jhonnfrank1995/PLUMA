# skills-lg — Stack de pensamiento LG Labs · Edición PLUMA Engine v2.0
Base: los 14 skills canónicos de LG Dev Studio. Esta copia NO es verbatim: es la edición del proyecto (ver CHANGELOG-EDICION-PLUMA.md) — núcleo universal intacto + secciones «Dirección en PLUMA Engine» + registro del ecosistema reescrito para este producto.

Instalación: copiar a `.claude/skills/` del repo pluma-engine (recomendado para esta edición: es por-proyecto por diseño, contiene direcciones específicas).
Si además se desea el stack genérico global para toda la empresa, instalar la versión CANÓNICA (sin direcciones) desde el repositorio de skills de LG en `~/.claude/skills/` — nunca esta edición, para no contaminar otros proyectos con direcciones de PLUMA.

Política de sincronización: mejoras al NÚCLEO (procesos, reglas) se aportan al canónico y se re-derivan aquí; las secciones «Dirección en PLUMA Engine» y el ecosystem-registry son locales del proyecto y evolucionan con él (revisión al abrir cada Etapa, comando /nueva-etapa).
